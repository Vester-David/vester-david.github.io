Topic 10 - HTML 5 Tags - Video, Audio and Canvas

HTML 5 allows you to embed video, audio and image files with simple tags.

<Video> tag attributes

  • src
  • controls
  • autoplay
  • width
  • height
  • loop
  • muted
  • poster="image.jpg" thumbnail
  • preload

An example tag for video:

<video src="microburst.mp4" width="500" controls autoplay loop muted></video>

You can also have fall back options for a video if the browser doesn't support a video format or HTML 5.

			<video controls>
			<source src="kitties.mp4" type="video/mp4; codecs='avc1, mp4a'">
			<source src="kitties.webm" type="video/webm; codecs='vp8.0, vorbis'">
			<p>Your browser does not support the video tag.</p>
			</video>

<Audio> tag

Audio files can be embedded in the same way as video files using the src, controls, autoplay and loop attributes to give the user options on how they can control the audio file.

An example tag for audio:

<audio src="amendrumbreak.mp3" controls ></audio>

Drum Break called "Amen"

Canvas tag

the Canvas tag specifies an "area" of space on the page that can have images, lines, shapes, gradients, text and more drawn on to it through Javascript code. By itself, the Canvas space us useless without input on what various events can do. You can wait for mouse clicks or movements to do various things or simply run javascript code that can be used to draw things by algorithm or coordinate. You can animate shapes or lines and manipulate pixels like converting a color image to black and white.

A circle drawn on a canvas element:

<canvas id="draw" width="100" height="100"></canvas>
<script>
var c = document.getElementById("draw");
var ctx = c.getContext("2d");
ctx.beginPath();
ctx.arc(50,50,40,0,2*Math.PI);
ctx.stroke();
</script>
