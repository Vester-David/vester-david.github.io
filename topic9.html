Topic 9 - Javascript Events

Types of Javascript Events:

  • Mouse Events
  • Keyboard Events
  • Frame/Object Events Events
  • Form Events
  • Drag Events
  • Clipboard Events
  • Print Events
  • Media Events
  • Mobile Touch Events

If the user can interact with it, you can pretty much capture what they did and perform various actions on that interation.

Can be handled inline through "button onclick="alert('you clicked!')" or "button onclick="myFunction()", or through addEventListener() in Javascript:

document.getElementById("myBtn").addEventListener("click", function(){
document.getElementById("demo").innerHTML = "Hello World";});

Source: https://www.w3schools.com/jsref/dom_obj_event.asp
