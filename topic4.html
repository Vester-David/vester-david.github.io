Topic 4 - Using XMLHTTPRequest to Consume a JSON Web Service

XMLHTTPRequest info

Example 1

XMLHTTPRequest is a behind-the-scenes request/answer system used to update a page with information without reloading the page. It's grouped in the AJAX family of techniques to access web servers from a webpage and update the DOM in the browser.

XMLHTTPRequest has a handshake process that starts by waiting for the XMLHTTPRequest object's state change to be ready, then making sure the server is ready to recieve a request by checking the ready state and status code sent from the server, sending the request to the server and then getting the response text from the server.

You can request data from text files, strings in JSON form and API web services.
