Topic 3 - JSON Parse, Stringify

JSON Parse

Example 1

JSON Parse takes a "JavaScript Object Notation" string and converts it into a Javascript Object.

JSON is a string format that shares information in name:value pairs much like arrays and objects.

An example JSON string is '{ "make":"Toyota", "model":"Tacoma", "color":"Forest Green", "type":"TRD PreRunner"}'

JSON Stringify

Example 2

Stringify does exactly the opposite of JSON Parse and converts a Javascript Object back into JSON string notation.
